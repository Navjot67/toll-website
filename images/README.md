# Example Images Directory

Place your example images here:

1. **ny-toll-account-example.jpg** - An example image showing where to find the NY Toll Bill Account Number
2. **nj-violation-example.jpg** - An example image showing where to find the NJ Toll Violation Number

## Supported Image Formats
- JPG/JPEG
- PNG
- WebP

## Image Naming
Make sure your images are named exactly:
- `ny-toll-account-example.jpg` (or .png, .webp)
- `nj-violation-example.jpg` (or .png, .webp)

## Tips
- Use clear, high-quality images
- Highlight or circle the relevant information in the images
- Keep file sizes reasonable (under 2MB recommended)
